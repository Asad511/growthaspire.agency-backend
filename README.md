# Growth Aspire Backend

This is the backend server for the Growth Aspire contact form. It uses Node.js, Express, and Nodemailer to handle form submissions and send email notifications.

## How to Run

1. Clone the repository
2. Run `npm install`
3. Create a `.env` file with:
   - EMAIL_USER
   - EMAIL_PASS
   - PORT (optional)
4. Run the server:

```bash
node server.js
